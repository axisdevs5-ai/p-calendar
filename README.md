## 🇮🇷 معرفی (فارسی)

**P-Calendar** یک ویجت مدرن و کاربردی است که علاوه بر نمایش تاریخ شمسی، امکاناتی مثل مدیریت رویدادها، اعلان‌های دسکتاپ و نمایش آب‌وهوا را هم در اختیار شما قرار می‌دهد.

### ✨ ویژگی‌ها

* 📅 نمایش تقویم کامل شمسی با محاسبهٔ سال‌های کبیسه.
* 🎨 طراحی مدرن با سه تم: روشن، تاریک و شیشه‌ای.
* 📝 مدیریت رویدادها (افزودن، ویرایش، حذف).
* 🔔 اعلان‌های سیستمی برای یادآوری رویدادها.
* 🌦️ نمایش وضعیت آب‌وهوا و ساعت ایران.
* 🚀 اجرای مستقیم به صورت **AppImage** بدون نصب اضافی.
* ⚡ رابط کاربری سبک و واکنش‌گرا.

---

## 📦 نسخه‌ها و توضیحات AppImage

در بخش [Releases](https://github.com/axisdevs5-ai/p-calendar/releases) می‌توانید نسخه‌ها را دریافت کنید.

* **نسخه‌های معمولی (حجم بالاتر):** اجرای اولیه سریع‌تر است (چون کمتر فشرده شده‌اند).
* **نسخه‌های فشرده‌تر (پسوند `-r`):** حجم کمتر دارند، اما اجرای اولیه ممکن است کمی طولانی‌تر باشد.

> پس از اجرای اولین بار، کش ایجاد می‌شود و دفعات بعدی سریع‌تر اجرا خواهند شد.

## 🚀 نصب و اجرا

### 📂 روش گرافیکی (دوبار کلیک)

1. فایل `.AppImage` را دانلود کنید.
2. روی فایل راست‌کلیک → Properties → Permissions.
3. تیک **Allow executing file as program** را بزنید.
4. فایل را با دوبار کلیک اجرا کنید ✅.

### 💻 روش ترمینال

```bash
chmod +x PCalendar-*.AppImage
./PCalendar-*.AppImage
```
